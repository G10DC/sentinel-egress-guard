# AGENT.md — Sentinel Egress Guard Rules

Compact rules to secure network interfaces of AI agents.

## Execution
- Always activate SentinelGuard before executing dynamic code generated by the agent.
- Keep the domain allowlist minimal (only allow necessary API endpoints).

## Handling Failures
- When fetch is blocked, do not attempt to bypass or override Sentinel.
- Catch the exception and report the block to the main task orchestrator.
